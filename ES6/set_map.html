ES6 Map and Set Examples

Map Examples

A Map holds key-value pairs where keys can be any datatype.

// Creating a Map const userMap = new Map(); // Setting values userMap.set('name', 'Alice'); userMap.set('age', 30); userMap.set('isAdmin', true); // Getting values console.log(userMap.get('name')); // Alice // Checking size console.log(userMap.size); // 3 // Iterating userMap.forEach((value, key) => { console.log(`${key}: ${value}`); }); // Converting to array console.log([...userMap.entries()]);

Set Examples

A Set stores unique values of any type.

// Creating a Set const numberSet = new Set(); // Adding values numberSet.add(1); numberSet.add(2); numberSet.add(3); numberSet.add(2); // Duplicate - won't be added // Checking size console.log(numberSet.size); // 3 // Checking for values console.log(numberSet.has(2)); // true // Iterating numberSet.forEach(value => { console.log(value); }); // Converting to array console.log([...numberSet]);
