ES6 Spread Operator and Rest Parameter

Spread Operator (...)

The spread operator allows you to spread out elements of an iterable object such as an array.

const odd = [1,3,5,7,9]; const even = [2,4,6,8,10]; const all = [...odd, ...even]; //concatenate arrays document.write("all: " + JSON.stringify(all) + "<br>"); const combined = [100,200,300, ...even]; //... operator unpacks array document.write("combined: " + JSON.stringify(combined) + "<br>"); const combined1 = [...odd, 333, 222, 111]; document.write("combined1: " + JSON.stringify(combined1) + "<br>");

Rest Parameter (...)

The rest parameter packs elements into an array. It's used in functions to accept variable number of arguments.

function display(num1, num2, ...args) { document.write("num1: " + num1 + ", num2: " + num2 + ", args: " + JSON.stringify(args) + "<br>"); } display(300, 400, 66, 77, 88, 99, 1000); display(30, 40, 66, 80, 54); display(10, 20);
